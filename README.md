# CRUD en PHP y MySQL - Sistema de Personajes

## Descripción del Proyecto
Este proyecto es una aplicación web funcional que implementa las cuatro operaciones básicas de gestión de bases de datos (CRUD: Crear, Leer, Actualizar y Eliminar). Está desarrollado utilizando **PHP** puro y conectado a una base de datos **MySQL**.

Para garantizar un código limpio y escalable, el proyecto está estructurado bajo el patrón de arquitectura **MVC** (Modelo-Vista-Controlador), separando de forma eficiente la conexión a la base de datos, la lógica del negocio y la interfaz de usuario. Además, el frontend está diseñado utilizando las clases y componentes de **Bootstrap** para lograr una interfaz moderna e intuitiva.

La entidad gestionada en este sistema es **Personajes** (un catálogo de personajes de videojuegos), la cual incluye los siguientes campos en su tabla:
- ID del personaje (Llave primaria auto-incrementable)
- Nombre
- Juego
- Elemento
- Rareza
- Nivel
